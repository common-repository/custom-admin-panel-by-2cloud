== Description ==

Add a custom admin panel with widget to your theme. Add your own logo, favicon, social media links, custom CSS and custom JS.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.You can also upload the zip file from within WordPress admin area. Plugins > Add New
== Frequently Asked Questions ==

= What does this plugin do? =

It installs an area in your WordPress admin where you can add custom logos, favicon, social media icons and custom CSS / JS to your theme.
